Compression

ORIGINAL

The original file (taken with my DSLR) is in .MOV format with a filesize of about 28MB. However, to speed up load times the videos were uploaded to YouTube and embedded into this page. YouTube uses a propriety and unpublished compression algorithm therefore, I could not find any reliable sources on how these videos are compressed after they are uploaded. It is worth mentioning that the videos on this page have double compression, the video format that I rendered them in and the format that YouTube uses. However, for the sake of this exercise, we will asume the YouTube compression is non-existant.

Video Format: H.264 | File Format: .MOV | File Size: 28.2MB | Compression Level: Original

QUALITY 10

For the next level I chose the DNxHR format. DNxHR (Digital Nonlinear Extensible High Resolution) is a lossy codec however, it is the least lossy available to me without using the original file. DNxHR was developed for 2K resolutions and up. When rendering a 1080p video, quality loss is not high but file sizes are large. DNxHR also uses the .mxf (Material Exchange Format) file format. On mac I could also have had the option to use the ProRes format which is considered less lossy than DNxHR however, as I do most of my work on Windows, that wasn't available to me.

Video Format: DNxHR | File Format: .mxf | File Size: 254MB | Quality Level: 10

QUALITY 7

The .mp4 file format is one of the most common because it was initallity popularised by Apple. Although it tends to be lower definition than other formats it is widely supported by almost all devices, mp4 supports the H.264 and MPEG-4 codecs. The H.264 codec is also one of the most commonly used because it can support resolutions up to 8K and can be wrapped with .mp4. This makes it easy to render high resolution video for almost all devices (many high-res formats are not widely supported, e.g. mxf). I used this combination for level 7 because I find that the quality to file size is satisfactory for the resolution and fps that I filmed in (1080p24fps). At 1080p the quality difference between level 7 and level 10 (above) is minimal but the file sizes are drastically different.

Video Format: H.264 | File Format: .mp4 | File Size: 4.3MB | Quality Level: 7
